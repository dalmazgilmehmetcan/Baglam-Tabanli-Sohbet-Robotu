1) "pip install -r requirements.txt" komutunu dosya dizininde açmış olduğunuz komut işlemi(cmd) üzerinde çalıştırınız.
2) "python chat.py" komutunu dosya dizininde açmış olduğunuz komut işlemi(cmd) üzerinde çalıştırınız. Chatbot çalışmış olacaktır.
3) Eğer eğitim gerçekleştirmek istiyorsanız "data.pth" eğitilmiş model dosyasını kaldırarak "python train.py" komutunu çalıştırınız.
